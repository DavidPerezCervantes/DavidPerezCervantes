## 🧠 Sobre mí

Soy estudiante de 8.º semestre de **Actuaría** (UAEMex) con una fuerte formación técnica previa en **Mecatrónica**. Mi perfil es un híbrido entre el rigor matemático tradicional y la agilidad algorítmica del desarrollo moderno. Me apasiona encontrar anomalías en los datos, automatizar flujos de trabajo y construir modelos que cuenten la historia real detrás de los números.

- 🔭 **Actualmente enfocado en:** Análisis de datos transaccionales, operaciones financieras (PayOps) y visualización de datos.
- 🌱 **Consolidando mi stack:** Escalando mi lógica de R hacia flujos automatizados con Python (Pandas) y extracciones complejas en SQL (`JOINs`).
- ⚡ **Dato curioso:** Mi lógica de programación nació arreglando sistemas de impresión 3D; hoy la utilizo para ejecutar inferencia Bayesiana e identificar discrepancias en grandes volúmenes de datos.
- 🤖 **Copiloto IA:** Utilizo Prompt Engineering para estructuración de scripts, agilizando la limpieza de datos y optimización de código.
